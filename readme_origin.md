项目说明：

eda_tools_cli.exe 和 eda_tools_ui.exe

- eda_tools_cli.exe





--file，        指定要处理的文件所在路径，必选
--file_suffix， 指定处理完毕后生成文件的后缀，默认为'_HybrdChip'，可选
--PCIE，        是否处理PCIE，可选
--GTP，         是否处理GTP，可选
--CRC，         是否计算CRC，可选
--COMPRESS，    是否压缩码流，可选
--TRIM，        是否修改trim寄存器，可选
--DELETE_GHIGH，是否移除ghigh，可选
